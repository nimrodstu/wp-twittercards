== Description ==

Add twitter cards to single posts and pages.

Select type of card.

Set default image to be used if no suitable image is found.

Set twitter username.

Add a twitter field to user profiles for use with multiple authors.


== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Settings > Wp Twitter Cards to set options
